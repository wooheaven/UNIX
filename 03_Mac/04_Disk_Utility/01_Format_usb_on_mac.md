# Format usb on mac
```
CMD + Space (find)
    -> disk utility
    -> Erase (지우기=지우면서 포맷하기)
        -> Name (usb 이름)
        -> Format
            -> ExFAT
        -> Scheme
            -> GPT or MBR ( GPT is more modern, MBR is older
    -> Erase (지우기)
Finder
    -> Eject "USB Drive"
```
